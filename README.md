# dev-java-cli-sbt-derby-simple

## Description
A java sbt build, that connects to derby
database.

## Tech stack
- java
- sbt
- derby driver

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
